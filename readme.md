
# Custom Cookie Content

A WordPress plugin that enables dynamic content management based on cookie values.

## Features

- Display content conditionally based on cookies
- Easy-to-use shortcodes
- Lightweight and performant

## Installation

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin in WordPress admin panel
3. Use shortcodes in your posts/pages

## Usage

```
[cookie_content cookie_name="value"]
    Your content here
[/cookie_content]
```

## Requirements

- WordPress 5.0 or higher
- PHP 7.2 or higher
